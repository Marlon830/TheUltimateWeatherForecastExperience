## Project Description
The Ultimate Weather Forecast Experience is a virtual reality (VR) application developed using Unity. This project aims to provide users with an immersive and interactive way to explore and experience weather forecasts. Users can visualize weather conditions in a dynamic and engaging 3D environment, making weather information more accessible and enjoyable.

## Features
- Interactive elements to explore different weather conditions (rain, snow, storm, clear and cloudy) with visual and audio feedback.
- The User can choose a specific day and time to see the corresponding weather forecast (within 7 days).
- User-friendly interface designed for VR headsets.

## Movement Method
The chosen method of movement within the VR environment is **teleportation**. This method ensures a comfortable and intuitive navigation experience, minimizing motion sickness for users.

## Interaction Method
The interaction methods implemented in the project are **Grab Interactable** object (the sphere) with a **Direct Interactor**. User can grab the sphere and rotate it from right to left and from left to right with both hands, the sphere will rotate but not move. User can also interact with the UI to change the weather conditions and time of day with a **Ray Interactor**.

## Requirements
- Unity 2021.3 or later.
- A VR headset compatible with Unity XR (e.g., Oculus, HTC Vive, etc.).

## Installation
1. Clone the repository to your local machine.
2. Open the project in Unity.
3. Ensure the required VR headset is connected and configured.
4. Build and run the project to start exploring the Ultimate Weather Forecast Experience.
